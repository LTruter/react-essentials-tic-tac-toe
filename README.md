# Tic Tac Toe - React Essentials Deep Dive

This repository is a completed learning project for the **React Essentials – Deep Dive** section. The goal of this project was to build a fully functional Tic Tac Toe game while reinforcing and expanding core React concepts.

## Goals

- Apply React fundamentals in a real interactive project
- Build and compose reusable components
- Manage component state effectively for game logic
- Handle user input and conditional rendering
- Learn best practices for state management and immutability
- Experiment with lifting state, computed values, and multi-component communication

## Key Concepts Explored

- **Components & JSX**: Splitting UI into reusable components, understanding isolation and instances
- **Props & Default Values**: Passing data down the tree and setting default prop values
- **State Management**: Using `useState`, updating state immutably, avoiding unnecessary state
- **Event Handling & User Input**: Two-way binding, button click handling, conditional button disabling
- **Conditional Rendering**: Showing game over screens, draw detection, dynamic content updates
- **Lists & Iteration**: Rendering the board using multi-dimensional arrays
- **Advanced Patterns**: Lifting state up, deriving state from props, sharing state across components
- **Project Organization**: Separating logic from UI, outsourcing static data, and improving maintainability

## Features

- Interactive two-player Tic Tac Toe game
- Dynamic board rendering with reusable square components
- Turn-based gameplay with accurate winner and draw detection
- Game reset and replay functionality
- Clean and responsive UI for better user experience

## How to Run

1. Clone this repo

   ```bash
   git clone https://github.com/LTruter/my-first-react-app.git

   ```

2. Install dependencies

   ```bash
   npm install

   ```

3. Start development server

   ```bash
   npm run dev

   ```

4. Open in browser

   Follow the URL shown in the terminal (usually http://localhost:5173/)
